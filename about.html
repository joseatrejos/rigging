Acerca de mí

A esto me dedico

Soy un ingeniero en producción multimedia, lo cual significa que sé producir productos con las últimas tecnologías de 3D, desarrollo, fotografía y diseño.

Entre las habilidades que que domino para desarrollar software y producciones increíbles se encuentran:

  • C# para desarrollo de videojuegos en Unity.
  • C++ para desarrollo de videojuegos en Unreal Engine.
  • Java para desarrollo en Android Studio.
  • Swift para desarrollo de aplicaciones iOS
  • Git para control de versiones.
  • HTML, CSS, Vanilla Javascript, PHP, Laravel y Django para diseño y desarrollo web.
  • Insomnia consumir y desarrollar APIs Rest.
  • Microsoft SQL Server y MySQL para bases de datos.
  • Maya, Zbrush, y Substance Painter para producción 3D.
  • Photoshop, Ilustrador e Indesign para diseño 2D y edición de imágenes.
  • Premiere Pro y Aftereffects para edición de video y corrección de color.
  • Adobe XD para prototipado de aplicaciones.
  • Adobe Media Encoder para renderizado de videos.
  • Fotografía e iluminación.

Además de todo esto, me encuentro impulsando mi empresa Ludens Productions, donde creamos, como equipo, software increíble y publicidad para negocios en forma de recorridos virtuales.
